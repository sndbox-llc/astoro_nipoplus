日付や時刻の入力に特化した入力フォームです。
日報を書く際に[提出日があらかじめ組み込まれています](/nipoplus/staff/writereport/#dist)が、それ以外で日付や時刻の入力が必要な場合はご利用下さい。

<dl class="basic">
  <dt>
    <a href="#point">日付と時刻（時点）</a>
  </dt>
  <dd>作業終了時刻や点検実施日時、トラブル発生時刻など、特定の一時点の入力に適しています</dd>
  <dt>
    <a href="#range">日付と時刻（期間）</a>
  </dt>

</dl>

## 日付と時刻（時点）

:::note[[入力イメージはこちらから確認できます](/nipoplus/reference/inputmethod/#datetime)]
:::

特定の出来事やイベントが発生した一時点の日付と時刻を記録するフォームです

### 日付と時刻（時点）フォームの主な用途

- 業務開始時間
- 業務終了時間
- 点検の完了時刻
- トラブル発生の時刻

定点的な記録に利用します。

<dl class="basic">
  <dt>集計</dt>
  <dd>❌ 非対応</dd>
  <dt>CSV出力</dt>
  <dd>✅ 対応</dd>
  <dt>PDF出力</dt>
  <dd>✅ 対応</dd>
  <dt>文字検索</dt>
  <dd>✅ 対応</dd>
</dl>

### 日付と時刻（時点）フォームの初期設定

<dl class="basic">
  <dt>入力必須</dt>
  <dd>ONの場合、日付や時刻が空欄だと日報の提出ができなくなります</dd>
  <dt>形式</dt>
  <dd>西暦を含めたり時刻を含めたりするかどうか、4種類から選択します。詳細な書式は以下の表を参照して下さい。</dd>
  <dt>その他の設定</dt>
  <dd>
    <a href="/nipoplus/editor/template/#common_setting">共通設定事項をご確認ください</a>
  </dd>
</dl>

形式の詳細は以下の通りです

|     入力形式     |   フォーマット   |        例        | 入力切替 |
| :--------------: | :--------------: | :--------------: | :------: |
| 日付（西暦有り)  |    YYYY/MM/DD    |    2022/12/01    |    ✅    |
| 日付（西暦なし） |      MM/DD       |      12/01       |    ❌    |
|     時刻のみ     |      HH:mm       |      13:21       |    ✅    |
|    日付と時刻    | YYYY/MM/DD HH:mm | 2022/12/01 13:21 |    ❌    |

### 日付と時刻（時点）のデータ集計

日付と時刻データはCSVに出力できます。出力は通常の文字列データとして行われます。

## 日付と時刻（期間）

開始時点〜終了時点」を入力することで、その間の期間（所要時間）を自動で計算し表示します。

### 日付と時刻（期間）フォームの主な用途

- 作業にかかった時間や日数を入力

<dl class="basic">
  <dt>集計</dt>
  <dd>✅ 対応</dd>
  <dt>CSV出力</dt>
  <dd>✅ 対応</dd>
  <dt>PDF出力</dt>
  <dd>✅ 対応</dd>
  <dt>文字検索</dt>
  <dd>✅ 対応</dd>
</dl>

### 日付と時刻（期間）フォームの初期設定

<dl class="basic">
  <dt>入力必須</dt>
  <dd>日付や時刻が空欄だと日報の提出ができなくなります</dd>
  <dt>集計する</dt>
  <dd>[集計機能](/nipoplus/staff/charts)で利用できます</dd>
  <dt>形式</dt>
  <dd>西暦を含めたり時刻を含めたりするかどうか、4種類から選択します。詳細な書式は以下の表を参照して下さい。</dd>
  <dt>スライダ上の最小（時刻のみ）</dt>
  <dd>スライダ式で時刻入力する際のスライダの左端を何時にするか設定します。初期値は8:00です</dd>
  <dt>スライダ上の最大（時刻のみ）</dt>
  <dd>スライダ式で時刻入力する際のスライダの右端を何時にするか設定します。初期値は20:00です</dd>
  <dt>刻み（時刻のみ）</dt>
  <dd>1目盛りあたり何分進めるか設定します。初期値は30分です</dd>
    <dt>その他の設定</dt>
  <dd>
    <a href="/nipoplus/editor/template/#common_setting">共通設定事項をご確認ください</a>
  </dd>

</dl>

入力形式の4種類については次のような形式になります。

|     入力形式     |   フォーマット   |        例        | 入力切替 |
| :--------------: | :--------------: | :--------------: | :------: |
| 日付（西暦有り)  |    YYYY/MM/DD    |    2022/12/01    |    ✅    |
| 日付（西暦なし） |      MM/DD       |      12/01       |    ❌    |
|     時刻のみ     |      HH:mm       |      13:21       |    ✅    |
|    日付と時刻    | YYYY/MM/DD HH:mm | 2022/12/01 13:21 |    ❌    |

日付と時刻（期間）では、入力した期間を下に自動で差分（経過した時間）を計算し画面に表示します。

:::tip[反復入力との組み合わせ]
[反復入力](/nipoplus/template/array)の中で日付と時刻（期間）を使い、かつ入力形式が「時刻のみ」の場合、時刻の始点が自動セットされる入力補助が発動します
:::

### 日付と時刻（期間）の集計

:::note[集計は「分単位」で行われます]
:::

期間入力フォームで記録された時間は集計が可能です。集計は単位が「分」に統一されることに注意してください。
例えば10:00 - 11:30は1時間半ですが、集計上は90分となります。[Excel活用テクニック](/excel/time/)も合わせてご覧下さい。

![期間のデータ（単位：分）を使って棒グラフを作成した例](./img/charts_tablet.png)

[日報のグラフ化についてはこちら](/nipoplus/reference/chart)
